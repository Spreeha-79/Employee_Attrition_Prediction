# HR Employee Attrition Analysis

## Project Overview
This project analyzes employee attrition patterns using HR data and machine learning models.

## Objectives
- Identify factors influencing employee attrition
- Analyze employee demographics and work characteristics
- Build predictive models to identify employees at risk of leaving
- Provide business recommendations to improve retention

## Dataset
IBM HR Employee Attrition Dataset

## Exploratory Data Analysis
Key areas analyzed:
- Monthly Income
- Work-Life Balance
- Department
- Job Role
- Overtime
- Years at Company

## Machine Learning Models
- Logistic Regression
- Random Forest
- XGBoost

## Results
The best-performing model was used to identify the most important factors associated with employee attrition.
